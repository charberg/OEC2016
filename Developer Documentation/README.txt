--Program Description--

This program is an interface to the Candy programming language.
A small simple language used to teach children the basics of programming using story telling.
This program will allow kids to create their own adventure using a simple intuitive interface.
By allows kids to create their own tale, we are introducing them to programming in a fun, easy way.

--Project Setup--

The project is available here: https://github.com/charberg/OEC2016
This project can be recreated and improved using any Java IDE. The IDE used during its creation was Eclipse Luna.
We strongly recommend using the Eclipse IDE as it was what the program was first developed in. Simply import the project through
the Eclipse program, and you will be on your way.
